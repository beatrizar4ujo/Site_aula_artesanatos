# Loja Artesanatos

## Overview
Loja Artesanatos is a website dedicated to showcasing various artisan classes and artworks. It serves as a platform for users to learn about different crafts, view completed projects, and get in touch for inquiries.

## Project Structure
The project consists of the following files:

- **index.html**: The main entry point of the website, containing the layout and structure of the homepage.
- **aulas.html**: Provides information about the classes offered, including details about instructors, schedules, and materials needed.
- **artes.html**: Showcases the artworks or crafts that have been created, including galleries or descriptions of completed projects.
- **contato.html**: Contains contact information and a form for users to reach out for inquiries or further information.
- **css/style.css**: Contains the styles for the website, defining the visual presentation of the HTML elements across the different pages.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project folder in your preferred web browser or code editor.
3. Open `index.html` to view the homepage and navigate through the other pages.
